# Wallpaper-App

Wallpaper Application


- Open Camera
- Check for Camera Permission
- If permission granted open Camera
- If permission not granted show alert dialog 
- Dont show again message in alert dialog 
- Capture image
- Set phone wallpaper
- App close automatically for 2 second
- Restart app again
